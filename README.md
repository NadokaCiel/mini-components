## 使用说明

### 指令说明

- 启动本地开发

微信小程序
```javascript
npm run dev
```

支付宝小程序
```javascript
npm run dev:aliapp
```

- 打包`production`生产环境代码

```javascript
npm run build
```

- 依赖包分析

```javascript
npm run report
```

- 使用`angular commit message`提交`git commit`信息

```javascript
npm run commit
```

- 安装`git hook`

```javascript
npm run husky
```

- 卸载`git hook`

```javascript
npm run unhusky
```
